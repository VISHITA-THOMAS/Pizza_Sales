# 🍕 Pizza Sales Analytics Dashboard – Power BI

## Project Overview

This project is an interactive Power BI dashboard designed to analyze pizza sales performance and customer ordering patterns. The dashboard provides insights into sales trends, customer preferences, peak ordering periods, and product performance to support data-driven business decisions.

The dataset was sourced from a publicly available online repository and underwent extensive data cleaning, transformation, and modeling before visualization.

---

## Objectives

- Analyze overall pizza sales performance.
- Identify the most popular pizza categories and sizes.
- Understand how order volumes vary across different days of the week.
- Explore the relationship between time periods and customer ordering behavior.
- Enable dynamic analysis through interactive slicers.

---

## Data Preparation

### Data Cleaning
- Removed inconsistencies and missing values.

- Validated date and time fields.
- Ensured data quality for accurate reporting.

### Data Modeling
- Established relationships between tables.
- Created a structured data model for efficient analysis.
